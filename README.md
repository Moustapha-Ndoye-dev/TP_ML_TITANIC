## 🏛️ Project Vision
This ecosystem covers the full MLOps lifecycle, from data processing and model versioning to real-time telemetry and data drift monitoring.

### 🧠 Explainable AI (XAI)
Unlike "black-box" models, this platform explains **why** a decision was made. 
- **Social Influence**: Impact of passenger class and ticket fare.
- **Profile Influence**: Impact of age, gender, and family status.
- **Proximity Influence**: Impact of cabin location and embarkation.

### 🛡️ MLOps Governance
- **Real-time Telemetry**: Monitoring CPU, RAM, and Latency.
- **Data Drift Detection**: Automated comparison between training distributions and live inference data.
- **Secure Architecture**: Built-in rate limiting, security headers, and request validation.
